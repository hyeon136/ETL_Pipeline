### ETL -> Extract, Transform, Load의 축약어로 data 분석을 위해 data warehouse, data mart 내로 이동시키는 process
* Extract : 소스 data로부터 추출
* Transform : DeNomalize 등의 추출된 데이터 변형
* Load : DW(DataWarehouse)로의 데이터 적재
